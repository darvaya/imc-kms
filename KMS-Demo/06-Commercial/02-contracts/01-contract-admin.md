# Contract Administration

## Purpose

This document covers the administration of commercial contracts.

---

## Contract Lifecycle

```
┌───────────┐   ┌───────────┐   ┌───────────┐   ┌───────────┐   ┌───────────┐
│ NEGOTIATE │──►│   SIGN    │──►│  EXECUTE  │──►│  MONITOR  │──►│   CLOSE   │
└───────────┘   └───────────┘   └───────────┘   └───────────┘   └───────────┘
```

---

## Contract Register

All contracts should be recorded:

| Field | Information |
|-------|-------------|
| Contract ID | Unique reference |
| Counterparty | Customer/supplier name |
| Type | Charter, COA, service, etc. |
| Start/End date | Contract period |
| Value | Contract amount |
| Key terms | Important provisions |
| Status | Active, expired, etc. |

---

## Key Obligations Tracking

| Obligation Type | Track |
|-----------------|-------|
| **Performance** | Loading rates, volumes |
| **Financial** | Payments, invoicing |
| **Reporting** | Required reports |
| **Renewal** | Notice periods |

---

## Performance Monitoring

| Monitor | Frequency |
|---------|-----------|
| Volume vs commitment | Monthly |
| Rate realization | Per shipment |
| Laytime/demurrage | Per voyage |
| Customer satisfaction | Periodic |

---

## Amendments

| Step | Action |
|------|--------|
| 1 | Identify need for change |
| 2 | Negotiate with counterparty |
| 3 | Document in writing |
| 4 | Sign amendment |
| 5 | Update contract register |

---

## Disputes

| Step | Action |
|------|--------|
| 1 | Identify issue |
| 2 | Gather documentation |
| 3 | Attempt commercial resolution |
| 4 | Escalate to management |
| 5 | Legal involvement if needed |
| 6 | Arbitration per contract |

---

## Contract Expiry

| Action | Timing |
|--------|--------|
| Review performance | Before expiry |
| Decide renewal | Per strategy |
| Negotiate if renewing | In advance |
| Close if not renewing | At expiry |

---

## Document Retention

| Document | Retention |
|----------|-----------|
| Signed contracts | 10 years after expiry |
| Amendments | With original |
| Correspondence | 7 years |
| Invoices | Per tax requirements |
