# Chartering Process

## Purpose

This document describes the chartering negotiation and fixture process.

---

## Process Overview

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   ENQUIRY   │───►│ NEGOTIATION │───►│   FIXTURE   │───►│  EXECUTION  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

---

## Stage 1: Enquiry

### Receiving Enquiries

| Source | Description |
|--------|-------------|
| **Direct** | From existing customers |
| **Broker** | Through chartering brokers |
| **Tender** | Response to RFQ/tender |

### Information Required

| Information | Purpose |
|-------------|---------|
| Cargo | Type, quantity |
| Route | Load/discharge ports |
| Laycan | Required delivery window |
| Rate indication | Budget/expectations |

---

## Stage 2: Negotiation

### Key Terms

| Term | Negotiation |
|------|-------------|
| **Rate** | Price per unit |
| **Laycan** | Delivery window |
| **Laytime** | Allowed time for cargo ops |
| **Demurrage/Dispatch** | Penalty/bonus rates |
| **Terms** | Payment, agency, etc. |

### Negotiation Process

| Step | Action |
|------|--------|
| 1 | Receive enquiry |
| 2 | Check availability |
| 3 | Calculate costs |
| 4 | Prepare offer |
| 5 | Negotiate terms |
| 6 | Agree main terms ("subjects") |

---

## Stage 3: Fixture

### Subjects

Common subjects (conditions) to lift:

| Subject | Meaning |
|---------|---------|
| **Sub board** | Management approval |
| **Sub shipper** | Shipper acceptance |
| **Sub details** | Final cargo details |
| **Sub survey** | Cargo quality |

### Fixture Recap

| Element | Included |
|---------|----------|
| Vessel name | Identified vessel |
| Cargo | Description, quantity |
| Loading | Port, rate, laytime |
| Discharge | Port, rate, laytime |
| Rate | Agreed freight |
| Terms | C/P proforma |

### Charter Party

Full contract based on standard forms with negotiated amendments.

---

## Stage 4: Execution

### Pre-Voyage

| Action | Responsibility |
|--------|----------------|
| Nominate vessel | Commercial |
| Advise ETA | Operations |
| Arrange agent | Commercial |

### During Voyage

| Action | Responsibility |
|--------|----------------|
| Monitor performance | Operations |
| Laytime tracking | Operations |
| Issue resolution | Commercial |

### Post-Voyage

| Action | Responsibility |
|--------|----------------|
| Laytime calculation | Commercial |
| Demurrage/Dispatch | Commercial |
| Invoice | Finance |
