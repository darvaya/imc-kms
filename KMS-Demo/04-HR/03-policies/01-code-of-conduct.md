# Code of Conduct

## Purpose

This Code of Conduct sets the standards of behavior expected of all IMC Pelita Logistik employees, both at sea and ashore.

---

## Our Values – COMPASS

Our Code is grounded in the COMPASS values:
- **C**ollaboration
- **O**bservant (Mindfulness)
- **M**anaging Resources (Prudence)
- **P**ioneer (Add Value)
- **A**gile (Strategic Mindset)
- **S**tewardship

---

## Professional Conduct

### Integrity

| Expectation | Details |
|-------------|---------|
| **Honesty** | Truthful in all dealings |
| **Fairness** | Treat all fairly |
| **Transparency** | Open communication |

### Compliance

| Expectation | Details |
|-------------|---------|
| **Laws** | Comply with all applicable laws |
| **Regulations** | Maritime regulations |
| **Company policies** | Internal policies |

---

## Workplace Behavior

### Respect

| Expectation | Details |
|-------------|---------|
| **Dignity** | Treat all with respect |
| **No discrimination** | Based on any protected characteristic |
| **No harassment** | Zero tolerance |

### Safety

| Expectation | Details |
|-------------|---------|
| **Safety first** | Never compromise safety |
| **Report hazards** | Speak up about concerns |
| **Follow procedures** | Comply with safety rules |

---

## Prohibited Conduct

| Prohibition | Examples |
|-------------|----------|
| **Drugs/Alcohol** | Zero tolerance on vessels |
| **Harassment** | Bullying, sexual harassment |
| **Theft/Fraud** | Any dishonesty |
| **Violence** | Physical or verbal abuse |
| **Discrimination** | Any form |

---

## Reporting Violations

| Channel | Use For |
|---------|---------|
| **Supervisor** | First point of contact |
| **HR** | HR-related concerns |
| **DPA** | Safety concerns |
| **Confidential hotline** | Anonymous reporting |

---

## Consequences

Violations of this Code may result in disciplinary action up to and including termination.
