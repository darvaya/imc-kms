# Disciplinary Policy

## Purpose

This document outlines the disciplinary policy for handling misconduct.

---

## Principles

| Principle | Details |
|-----------|---------|
| **Fairness** | Consistent, fair treatment |
| **Proportionality** | Penalty matches offense |
| **Investigation** | Facts established before action |
| **Right to respond** | Employee can present their case |

---

## Categories of Misconduct

### Minor Misconduct

| Examples | Typical Action |
|----------|---------------|
| Lateness | Verbal warning |
| Minor policy breach | Written warning |
| Poor performance | Performance improvement |

### Serious Misconduct

| Examples | Typical Action |
|----------|---------------|
| Repeated minor misconduct | Final warning |
| Negligence | Written/final warning |
| Insubordination | Warning/dismissal |

### Gross Misconduct

| Examples | Typical Action |
|----------|---------------|
| Theft/fraud | Summary dismissal |
| Violence | Summary dismissal |
| Drug/alcohol abuse | Summary dismissal |
| Willful damage | Summary dismissal |
| Gross negligence | Summary dismissal |
| Harassment | Dismissal |

---

## Disciplinary Process

### Investigation

| Step | Action |
|------|--------|
| 1 | Allegation received |
| 2 | Preliminary assessment |
| 3 | Investigation if warranted |
| 4 | Gather evidence, interview |

### Hearing

| Step | Action |
|------|--------|
| 1 | Employee notified in writing |
| 2 | Employee can prepare response |
| 3 | Disciplinary hearing held |
| 4 | Employee presents case |
| 5 | Decision made |

### Outcomes

| Outcome | When Applied |
|---------|--------------|
| **No action** | Allegation not substantiated |
| **Verbal warning** | Minor first offense |
| **Written warning** | Recorded warning |
| **Final warning** | Serious or repeated |
| **Demotion** | Where appropriate |
| **Dismissal** | Gross or repeated misconduct |

### Appeal

| Right | Details |
|-------|---------|
| **Timeframe** | Within 7 days |
| **To** | Higher authority |
| **Process** | Review of decision |

---

## Records

| Requirement | Details |
|-------------|---------|
| **Warnings** | Kept on file |
| **Duration** | Per policy (e.g., 12 months) |
| **Confidentiality** | Maintained |
