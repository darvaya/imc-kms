## How to Request Support

### Step 1: Submit Request

| Information | Why Needed |
|-------------|------------|
| Your name & contact | To reach you |
| Issue description | Understand problem |
| System/application | What's affected |
| Error messages | Diagnosis |
| Urgency | Prioritization |

### Step 2: Ticket Created

Your request is logged and assigned a ticket number.

### Step 3: Response

| Priority | Response Time |
|----------|---------------|
| **Critical** | < 1 hour |
| **High** | < 4 hours |
| **Medium** | < 1 business day |
| **Low** | < 3 business days |

### Step 4: Resolution

IT works to resolve your issue and will update you on progress.

---

## Priority Definitions

| Priority | Definition | Examples |
|----------|------------|----------|
| **Critical** | Business stopped | Server down, email outage |
| **High** | Major impact | Can't access key system |
| **Medium** | Work affected | Software issue |
| **Low** | Minor/request | Questions, requests |

---

## Self-Help

Before contacting IT, try:
- Restart computer
- Check network connection
- Search this KMS for guides
- Ask a colleague

---

## Escalation

If issue not resolved satisfactorily:
1. Request escalation with IT
2. Contact IT Manager
