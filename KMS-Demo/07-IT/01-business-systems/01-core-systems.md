# Core Business Systems

## Purpose

This document provides an overview of core business systems at IMC Pelita Logistik.

---

## Operations Systems

### Planned Maintenance System (PMS)

| Aspect | Details |
|--------|---------|
| **Purpose** | Vessel maintenance scheduling |
| **Features** | Job scheduling, work orders, history |
| **Users** | Technical, vessel officers |
| **Access** | Desktop + vessel terminals |

### Crew Management System

| Aspect | Details |
|--------|---------|
| **Purpose** | Seafarer records and scheduling |
| **Features** | Personnel records, certification, rotation |
| **Users** | Crewing department |
| **Access** | Desktop application |
