# Information Security

## Purpose

This document outlines information security principles and practices.

---

## Security Objectives

| Objective | Description |
|-----------|-------------|
| **Confidentiality** | Information accessed only by authorized persons |
| **Integrity** | Information accurate and complete |
| **Availability** | Information available when needed |

---

## Data Classification

| Classification | Examples | Handling |
|----------------|----------|----------|
| **Confidential** | Financials, contracts, personal data | Restricted access, encryption |
| **Internal** | Policies, procedures, reports | Employees only |
| **Public** | Website content, published info | No restriction |

---

## Security Responsibilities

### All Employees

| Responsibility | Action |
|----------------|--------|
| **Passwords** | Keep secure, don't share |
| **Devices** | Lock when unattended |
| **Data** | Handle per classification |
| **Report** | Report incidents |

### IT Department

| Responsibility | Action |
|----------------|--------|
| **Systems** | Maintain security controls |
| **Access** | Manage user access |
| **Monitoring** | Monitor for threats |
| **Response** | Respond to incidents |

---

## Access Control

| Principle | Application |
|-----------|-------------|
| **Least privilege** | Minimum necessary access |
| **Need to know** | Based on role |
| **Separation of duties** | Prevent conflicts |
| **Regular review** | Periodic access audit |

---

## Physical Security

| Control | Location |
|---------|----------|
| **Access cards** | Office building |
| **Locked rooms** | Server rooms |
| **Secure storage** | Sensitive documents |

---

## Cybersecurity

| Threat | Protection |
|--------|------------|
| **Malware** | Antivirus, updates |
| **Phishing** | Awareness training |
| **Ransomware** | Backups, patching |
| **Unauthorized access** | Strong passwords, MFA |

---

## Incident Reporting

Report security incidents immediately:
- Lost/stolen devices
- Suspected phishing
- Unauthorized access
- Data breaches

| Report to | How |
|-----------|-----|
| IT Helpdesk | Email or phone |
| IT Manager | Urgent matters |
