# Mooring Operations Safety

## Purpose

This document provides safety procedures for mooring operations across all IMC Pelita Logistik vessels to prevent injury from mooring line failures and equipment hazards.

---

## Scope

Applies to: **All vessel types and FLF**

---

## Mooring Hazards

| Hazard | Description | Consequence |
|--------|-------------|-------------|
| **Snapback** | Parted line recoils with lethal force | Death, serious injury |
| **Crush** | Trapped between equipment or vessels | Amputation, death |
| **Entanglement** | Caught in running line | Amputation, drowning |
| **Struck by** | Hit by thrown heaving line or equipment | Injury |
| **Falls** | Slippery deck, uneven surfaces | Injury |

---

## Snapback Zone

> ⚠️ **DANGER:** When mooring lines are under tension, they store enormous energy. If a line parts, it whips back violently through a "snapback zone."

### Definition
The snapback zone is the area where a parted line could strike – typically a cone extending from each attachment point (bitts, bollards, fairleads).

### Snapback Zone Identification

| Marking | Method |
|---------|--------|
| Deck marking | Yellow/black diagonal stripes or solid red |
| Signs | "DANGER – SNAPBACK ZONE" posted |
| Training | All crew trained to identify zones |

### Rule
**NO STANDING IN SNAPBACK ZONES** when lines are under tension – regardless of how briefly.

## Safe Mooring Procedures

### Before Mooring

| Action | Checked |
|--------|---------|
| Mooring plan communicated to all | ☐ |
| Lines inspected, in good condition | ☐ |
| Winches and capstans tested | ☐ |
| Deck clear of obstructions | ☐ |
| PPE donned (gloves, hard hat, safety boots) | ☐ |
| Communication established (radio/signals) | ☐ |

### During Mooring

| Requirement | Details |
|-------------|---------|
| **Line handling** | Never wrap line around hands/body |
| **Line of fire** | Stay outside line of fire at all times |
| **Turns on drum** | Proper number of turns before tensioning |
| **Heaving line** | Clear warning before throwing |
| **Tension** | Do not overtension lines |
| **Communication** | Continuous between deck and bridge |

### Standard Hand Signals

| Signal | Meaning |
|--------|---------|
| Arm rotating | Heave in |
| Hand moving outward | Pay out |
| Crossed arms | Stop |
| Thumbs up | All secure |
| Waving vigorously | Emergency stop |

## Mooring Line Inspection

### Synthetic Lines (Polypropylene, Polyester, HMPE)

| Check | Defect Indicators |
|-------|-------------------|
| Surface | Cuts, abrasion, glazing, fusion |
| Core | Inner yarns visible, broken strands |
| Stiffness | Hardened sections (heat damage) |
| Discoloration | Chemical damage, UV degradation |

### Wire Rope

| Check | Defect Indicators |
|-------|-------------------|
| Broken wires | >10% broken in one lay length |
| Corrosion | Rust, pitting |
| Kinks | Permanent deformation |
| Bird caging | Strands separated |

### When to Retire Lines

- Any defect affecting > 10% of strength
- Age exceeds manufacturer recommendation
- After any snapback incident (inspect carefully)
- After heavy shock loading

## Mooring Configurations

### Standard Pattern (Alongside)

```
                    QUAY/VESSEL
   ─────────────────────────────────────
         Headline       Headline
            \             /
             \           /
    Breast ───●─────────●─── Breast
              │         │
              │  SHIP   │
              │         │
    Spring ───●─────────●─── Spring
             /           \
            /             \
        Stern line    Stern line
   ─────────────────────────────────────
                    QUAY/VESSEL
```

| Line Type | Purpose |
|-----------|---------|
| **Headlines** | Prevent forward movement |
| **Stern lines** | Prevent aft movement |
| **Breast lines** | Prevent off-berth movement |
| **Springs** | Prevent fore/aft movement along berth |

## Mooring Winch Safety

| Requirement | Details |
|-------------|---------|
| **Guards** | All moving parts guarded |
| **Emergency stop** | Accessible at winch station |
| **Brakes** | Tested before operation |
| **Operator training** | Only trained personnel operate |
| **Communications** | Clear line of sight or radio |

---

## Securing for Departure

| Step | Action |
|------|--------|
| 1 | Master/pilot confirms ready |
| 2 | Single up to headlines/stern lines |
| 3 | Let go springs and breasts |
| 4 | Let go headlines and stern lines |
| 5 | Retrieve lines clear of water |
| 6 | Stow lines and secure equipment |

---

## Night Mooring

Additional precautions:
- Adequate deck lighting
- Reflective PPE worn
- Enhanced communication
- Experienced crew assigned
- Extra vigilance around equipment

---

## Incident Reporting

All mooring incidents must be reported:
- Near misses (line under high strain, near parting)
- Equipment failures
- Personal injury
- Line failures

Use [Incident Reporting](../08-incident-management/02-reporting-procedures.md) procedures.
