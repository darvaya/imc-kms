# Barge-to-FLF Transfer Procedures

## Purpose

This document provides safety procedures for the transfer of coal cargo from barges to the Floating Loading Facility (FLF).

---

## Scope

Applies to: **FLF operations and attending barges**

---

## Pre-Transfer Checklist

### FLF Preparation

| Item | Checked |
|------|---------|
| Conveyor systems pre-start inspected | ☐ |
| Emergency stops tested and functional | ☐ |
| Communication systems working | ☐ |
| Deck lighting adequate (if night ops) | ☐ |
| Fire extinguishers in place | ☐ |
| Crew briefed on operations | ☐ |
| Weather forecast reviewed | ☐ |
| Fendering in good condition | ☐ |

### Barge Preparation

| Item | Checked |
|------|---------|
| Hatch covers opened and secured | ☐ |
| Cargo accessible for discharge | ☐ |
| Mooring lines ready | ☐ |
| Communication with FLF established | ☐ |
| Crew briefed on safety | ☐ |
| Draft survey completed | ☐ |

---

## Approach and Mooring

### Barge Approach

| Step | Action |
|------|--------|
| 1 | Tug/barge contacts FLF on VHF |
| 2 | FLF confirms ready to receive |
| 3 | Approach at slow speed (< 2 knots) |
| 4 | Approach angle parallel to FLF |
| 5 | FLF deck crew ready with lines |
| 6 | Tug positions barge alongside fendering |

### Mooring Sequence

| Step | Action |
|------|--------|
| 1 | Pass headlines first |
| 2 | Pass stern lines |
| 3 | Pass breast lines |
| 4 | Tug stands clear (or assists as needed) |
| 5 | Adjust lines for even tension |
| 6 | Confirm mooring complete |

> See [Mooring Operations](./02-mooring-operations.md) for detailed procedures.

## Cargo Transfer Operations

### Starting Transfer

| Step | Action |
|------|--------|
| 1 | FLF Control confirms barge secured |
| 2 | Conveyor pre-start alarm sounded (3 times) |
| 3 | All clear received from deck |
| 4 | Conveyor started at slow speed |
| 5 | Grab/crane begins loading |
| 6 | Increase to normal operating speed |

### During Transfer

| Requirement | Details |
|-------------|---------|
| **Exclusion zones** | No personnel within grab swing radius |
| **Continuous monitoring** | Watch for spillage, conveyor issues |
| **Communication** | Radio contact maintained |
| **Trim** | Monitor barge trim during discharge |
| **Weather watch** | Stop if conditions deteriorate |

### Stopping Criteria

Stop cargo operations immediately if:
- ⚠️ Wind exceeds 25 knots
- ⚠️ Swell causes excessive movement
- ⚠️ Mooring line failure
- ⚠️ Conveyor malfunction
- ⚠️ Fire or smoke detected
- ⚠️ Man overboard
- ⚠️ Any emergency alarm

## Personnel Safety

### Exclusion Zones

| Zone | Restriction |
|------|-------------|
| Grab operating area | No entry during operations |
| Conveyor belt area | No entry without permit |
| Mooring line areas | Stand clear during tensioning |
| Between vessels | No crossing – use gangway only |

### PPE Requirements

| PPE | Required For |
|-----|--------------|
| Hard hat | All deck personnel |
| Safety boots (steel toe) | All deck personnel |
| High-visibility vest | All deck personnel |
| Dust mask (P2/N95) | When dust present |
| Safety glasses | Near cargo operations |
| Gloves | Mooring and line handling |
| Lifejacket | Available and accessible |

---

## Night Operations

Additional requirements for night transfer:

| Requirement | Details |
|-------------|---------|
| Adequate lighting | Deck, cargo areas, gangway |
| Navigation lights | As required by COLREGS |
| Increased vigilance | Extra lookouts posted |
| Reduced speed | Slower approach |
| Crew experience | Only experienced crew assigned |

---

## Completion and Departure

### Completing Transfer

| Step | Action |
|------|--------|
| 1 | Confirm cargo holds empty |
| 2 | Stop conveyor system |
| 3 | Clear any spillage on deck |
| 4 | Close barge hatch covers |
| 5 | Complete cargo documentation |

### Barge Departure

| Step | Action |
|------|--------|
| 1 | Tug returns alongside |
| 2 | Tug confirms ready to tow |
| 3 | Let go lines (springs, breasts, headlines) |
| 4 | Cast off stern lines last |
| 5 | Tug maneuvers barge clear |
| 6 | FLF confirms clear departure |
