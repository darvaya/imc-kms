# Cargo Temperature Monitoring

## Purpose

This document provides procedures for monitoring coal cargo temperature to detect self-heating before it leads to fire.

---

## Scope

Applies to: **Motor Vessels (Bulk Carriers) carrying coal cargo**

---

## Why Monitor Temperature?

Coal self-heating is a progressive process. Temperature monitoring allows early detection so that action can be taken before fire develops.

| Temperature Zone | Status | Action |
|------------------|--------|--------|
| < 40°C | Normal | Routine monitoring |
| 40-55°C | Elevated | Increased monitoring, prepare to seal |
| 55-70°C | Critical | Seal hold, seek advice, consider deviation |
| > 70°C | Danger | Emergency procedures, CO2 may be needed |
| > 150°C | Fire | Fire exists, full emergency response |

---

## Monitoring Methods

### 1. Fixed Temperature Sensors

| Feature | Details |
|---------|---------|
| Type | Thermocouples or resistance sensors |
| Location | Inserted into hold via sampling pipes |
| Reading | Bridge or Engine Control Room display |
| Advantage | Continuous, no entry required |

### 2. Portable Temperature Probes

| Feature | Details |
|---------|---------|
| Use | Insert through hold access/manholes |
| Depth | Probe into cargo at various depths |
| Advantage | Can check specific areas |
| Limitation | Requires access to hold |

### 3. Infrared Thermometer

| Feature | Details |
|---------|---------|
| Use | Surface temperature reading |
| Advantage | No contact required |
| Limitation | Only measures surface |

---

## Monitoring Procedure

### Frequency

| Phase | Standard | If Elevated |
|-------|----------|-------------|
| Loading | Every 4 hours | N/A |
| Sea passage | Every 8 hours | Every 4 hours |
| Approaching discharge | Every 4 hours | Every 2 hours |

### Measurement Points

For each cargo hold, measure at:
- Forward location (near bulkhead)
- Midships location (center of hold)
- Aft location (near bulkhead)
- At different depths if probe available

### Recording

| Data | Format |
|------|--------|
| Date/Time | DD-MMM-YYYY HH:MM |
| Hold number | 1, 2, 3, etc. |
| Location | Fwd/Mid/Aft |
| Temperature | °C |
| Trend | Rising/Stable/Falling |
| Sea water temp | °C (for comparison) |
| Signature | Officer recording |

---

## Response to Elevated Temperature

### If Temperature Rising Above 40°C:

| Step | Action |
|------|--------|
| 1 | Note rate of increase |
| 2 | Cross-check with CO readings |
| 3 | Inform Master |
| 4 | Increase monitoring frequency |
| 5 | Stand by to seal hold |

### If Temperature Exceeds 55°C:

| Step | Action |
|------|--------|
| 1 | CLOSE all ventilation to hold |
| 2 | Seal all openings |
| 3 | Inform Master and company/DPA |
| 4 | Prepare boundary cooling |
| 5 | Seek expert advice (Class, P&I) |
| 6 | Consider diversion to port |

### If Temperature Exceeds 70°C:

| Step | Action |
|------|--------|
| 1 | Assume fire is developing |
| 2 | Implement cargo hold fire procedures |
| 3 | Consider CO2/inert gas injection |
| 4 | Divert to nearest safe port |
| 5 | Request firefighting assistance at port |

---

## Interpreting Trends

| Observation | Meaning |
|-------------|---------|
| Slow, steady rise | Self-heating in progress |
| Rapid rise | Accelerated heating – urgent action |
| Stable elevated | Possible localized hot spot |
| Rising CO with rising temp | Confirms self-heating |
| Temperature falling | Situation stabilizing (if sealed) |

---

## Factors Affecting Readings

| Factor | Effect |
|--------|--------|
| **Solar heating** | Deck/hatch covers heat cargo surface |
| **Sea water temperature** | Affects hull and lower cargo |
| **Sensor location** | Hot spot may not be where sensor is |
| **Insulation** | Coal is a good insulator – core may be hotter |

> **Best Practice:** Use multiple measurement points and look for trends over time rather than individual readings.

---

## Pre-Loading Temperature Check

Before accepting coal cargo:

| Check | Action |
|-------|--------|
| Stockpile history | Any previous heating issues? |
| Temperature at the time of loading | Should be <40°C |
| Recent rainfall | Wet coal at higher risk |
| Time since mining | Freshly mined = more reactive |
