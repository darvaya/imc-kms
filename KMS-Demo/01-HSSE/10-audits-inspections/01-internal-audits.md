# Internal Audits

## Purpose

This document provides procedures for conducting internal Safety Management System (SMS) audits to verify compliance and identify improvement opportunities.

---

## Scope

Applies to: **All vessels, facilities, and shore-based operations**

---

## Audit Objectives

- Verify compliance with SMS procedures
- Verify compliance with regulations (ISM, ISPS, MARPOL, etc.)
- Identify non-conformities and areas for improvement
- Provide input for management review
- Demonstrate due diligence

---

## Audit Schedule

| Area | Frequency | Auditor |
|------|-----------|---------|
| Each vessel | Annual (minimum) | HSSE Manager/DPA |
| FLF operations | Annual | HSSE Manager |
| Shore operations | Annual | HSSE Manager |
| Specific processes | As needed | Designated auditor |

---

## Auditor Requirements

| Requirement | Details |
|-------------|---------|
| Training | Lead Auditor or Internal Auditor training |
| Independence | Cannot audit own work area |
| Knowledge | Familiar with SMS and regulations |
| Authority | Management support for access |

---

## Audit Process

### 1. Planning

| Activity | Details |
|----------|---------|
| Define scope | Which areas/processes |
| Review previous | Past audits, NCRs, incidents |
| Prepare checklist | Based on SMS, regulations |
| Notify auditee | At least 2 weeks notice |

### 2. Opening Meeting

| Agenda | Details |
|--------|---------|
| Introduce auditor(s) | |
| Confirm scope | |
| Explain process | |
| Agree schedule | |
| Request cooperation | |

### 3. Audit Execution

| Method | Application |
|--------|-------------|
| **Document review** | Certificates, records, permits |
| **Observation** | Actual practices vs. procedures |
| **Interviews** | Crew knowledge and understanding |
| **Physical inspection** | Equipment, housekeeping |

### 4. Findings Classification

| Finding Type | Definition |
|--------------|------------|
| **Major Non-Conformity** | Serious failure posing risk or absence of effective SMS element |
| **Non-Conformity (Minor)** | Single observed failure not posing immediate risk |
| **Observation** | Not a failure but opportunity for improvement |
| **Positive Finding** | Good practice worth recognizing |

### 5. Closing Meeting

| Agenda | Details |
|--------|---------|
| Present findings | Discuss each finding |
| Agree corrective actions | Responsibilities and timelines |
| Thank auditee | |
| Explain next steps | Report, follow-up |

### 6. Audit Report

| Section | Content |
|---------|---------|
| Executive summary | Overview of findings |
| Scope and criteria | What was audited |
| Findings | Listed with classification |
| Recommendations | For each finding |
| Corrective action plan | Responsibilities, deadlines |

---

## Corrective Action Management

| Severity | Response Time |
|----------|---------------|
| Major NC | Within 3 months |
| Minor NC | Before next audit |
| Observation | At discretion |

### Tracking

| Element | Details |
|---------|---------|
| NCR log | Register of all findings |
| Status | Open / In Progress / Closed |
| Verification | Evidence of completion |
| Effectiveness | Confirmed at next audit |
