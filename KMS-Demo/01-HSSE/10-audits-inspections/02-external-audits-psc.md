# External Audits & Port State Control

## Purpose

This document provides guidance for preparing and responding to external audits and Port State Control inspections.

---

## Scope

Applies to: **All vessels**

---

## Types of External Inspections

| Type | Authority | Purpose |
|------|-----------|---------|
| **Flag State Audit** | Indonesia (DGST) | ISM/ISPS verification |
| **Classification Survey** | Class Society | Statutory certificates |
| **Port State Control (PSC)** | Port State Authority | Compliance verification |
| **Vetting Inspection** | Charterer/Client | Commercial qualification |

---

## Port State Control (PSC)

### What is PSC?

PSC is an inspection by port state authorities to verify that foreign-flag vessels comply with international conventions (SOLAS, MARPOL, STCW, etc.).

### MOU Regions

Indonesia is part of the **Tokyo MOU** region.

| MOU | Region |
|-----|--------|
| Tokyo MOU | Asia-Pacific |
| Paris MOU | Europe |
| Indian Ocean MOU | Indian Ocean |

### Targeting

Vessels are selected for inspection based on:
- Flag State performance
- Classification Society performance
- Ship type and age
- Company performance
- Previous detention history
- Time since last inspection

## PSC Preparation

### Ongoing Readiness

| Area | Requirement |
|------|-------------|
| Certificates | Valid, displayed, readily available |
| Equipment | Operational, tested, maintained |
| Records | Oil Record Book, Garbage Record, logs current |
| Drills | Conducted as required, recorded |
| Deficiencies | Known defects documented with action plan |

| Item | Checked |

| Garbage Record Book up to date | ☐ |
| Crew certificates on board | ☐ |
| MARPOL equipment functional | ☐ |

## PSC Inspection Areas

| Area | Focus |
|------|-------|
| Certificates | All statutory and company |
| Crew documentation | CoCs, medicals, work/rest records |
| Structural | Hull, hatches, cargo holds |
| Safety equipment | LSA, firefighting, navigation |
| MARPOL equipment | OWS, ODME, garbage, sewage |
| ISM implementation | SMS, familiarization, drills |
| ISPS compliance | Security plan, access control |

---

## PSC Outcomes

| Outcome | Action |
|---------|--------|
| **No deficiencies** | Thank inspector, continue voyage |
| **Deficiencies noted** | Rectify within timeframe |
| **Detention** | Cannot sail until rectified |

### Detention Consequences

- Flag State notified
- Classification Society notified
- Company name published
- Increased targeting for future inspections
- Potential for charterer rejection

---

## Responding to Findings

| Finding Type | Response |
|--------------|----------|
| Rectify before departure | Fix immediately |
| Rectify at next port | Document, plan, execute |
| Detention condition | All resources to resolve |

### Documentation

All PSC inspections:
- Record in company database
- Notify DPA immediately
- Complete corrective actions
- Verify effectiveness
- Report to management
