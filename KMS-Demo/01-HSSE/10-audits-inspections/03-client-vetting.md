# Client Vetting

## Purpose

This document provides guidance on client vetting inspections, particularly for dry bulk operations where charterers may require vessel qualification.

---

## Scope

Applies to: **Motor Vessels on commercial charter**

---

## What is Vetting?

Vetting is a process where charterers (cargo owners, traders) assess whether a vessel meets their safety and quality standards before chartering.

For **dry bulk** (coal), the primary vetting systems are:
- **Rightship** with **RISQ** scoring
- **Client-specific inspections**

---

## Rightship & RISQ

### What is Rightship?

Rightship is a maritime risk management organization that provides:
- Safety scores for vessels
- Inspection management
- Due diligence services

### RISQ Score

| Grade | Description |
|-------|-------------|
| **1** | Highest performance |
| **2** | Above average |
| **3** | Acceptable |
| **4** | Needs improvement |
| **Unacceptable** | Fails to meet standards |

### Factors Affecting RISQ

| Factor | Impact |
|--------|--------|
| PSC history | Detentions hurt score |
| Casualty record | Incidents affect score |
| Vessel age | Older = potentially lower |
| Flag State | Flag performance matters |
| Class Society | Class performance matters |
| Company score | Overall company performance |
| Dry dock compliance | On-time dry docking |

## Preparing for Client Inspection

### Before Inspection

| Action | Responsibility |
|--------|----------------|
| Review previous inspection reports | Master |
| Conduct pre-inspection walk-through | C/O |
| Ensure certificates ready | Master |
| Housekeeping check | C/O |
| Equipment testing | All officers |
| Crew briefing | Master |

### Focus Areas

| Area | Key Items |
|------|-----------|
| Bridge | ECDIS, AIS, VDR, navigation equipment |
| Deck | Cranes, hatches, holds, mooring |
| Engine | Machinery condition, maintenance |
| Accommodation | Cleanliness, safety |
| Safety equipment | LSA, firefighting |
| Documentation | Certificates, records, SMS |

---

## During Vetting Inspection

| Do | Don't |
|----|-------|
| Be professional and welcoming | Be defensive |
| Demonstrate knowledge | Guess or make up answers |
| Show documentation promptly | Hide problems |
| Explain deficiencies honestly | Argue with inspector |
| Escort throughout | Leave inspector unaccompanied |

---

## After Inspection

| Action | Timeline |
|--------|----------|
| Receive verbal debrief | End of inspection |
| Receive formal report | Days to weeks |
| Create action plan | Within 48 hours of report |
| Close observations | Per agreed timeline |
| Report to company | Immediately |

---

## Managing Observations

| Response | Details |
|----------|---------|
| Agree | Accept and fix |
| Clarify | Provide additional information if misunderstood |
| Escalate | If disagree, discuss with company/DPA |

> **Priority:** Close observations promptly to maintain good vetting status.
