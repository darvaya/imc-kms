# Bunker Operations Safety

## Purpose

This document provides safety procedures for bunkering (fuel transfer) operations to prevent oil spills and ensure safe fuel handling.

---

## Scope

Applies to: **All vessel types receiving fuel**

---

## Bunker Hazards

| Hazard | Risk |
|--------|------|
| **Oil spill** | Environmental pollution, fines |
| **Overflow** | Tank overfill, deck spill |
| **Fire/explosion** | Vapor ignition |
| **Slip/fall** | Oily deck surfaces |
| **Injuries** | Hose handling, manual tasks |

---

## Pre-Bunkering Checklist

### Vessel Preparation

| Item | Checked |
|------|---------|
| Bunker plan prepared (tank sequence, quantities) | ☐ |
| Tank levels verified (room for planned intake) | ☐ |
| High-level alarms tested | ☐ |
| Overflow tanks available and empty | ☐ |
| Vents clear and functional | ☐ |
| Scuppers plugged | ☐ |
| Drip trays in place under manifold | ☐ |
| Fire extinguisher at manifold | ☐ |
| SOPEP equipment ready | ☐ |
| Communication agreed | ☐ |
| Emergency stop procedure agreed | ☐ |
| No smoking signs posted | ☐ |
| No hot work during bunkering | ☐ |

### Ship-Shore Safety Checklist

Complete the **Ship-Shore Safety Checklist** with bunker barge/terminal before starting.

| Section | Topics |
|---------|--------|
| General | Moorings, access, lighting |
| Fire safety | Fire-fighting ready, no hot work |
| Spill prevention | Scuppers plugged, drip trays |
| Communications | Agreement on stop signals |
| Emergency | Stop procedures, emergency contacts |

## During Bunkering

### Key Procedures

| Requirement | Action |
|-------------|--------|
| **Continuous watch** | Duty officer monitors throughout |
| **Tank monitoring** | Regular sounding of receiving tanks |
| **Topping off** | Slow rate for final 85% fill |
| **Communication** | Maintain contact with bunker barge |
| **No distractions** | Bunker duty takes priority |

### Monitoring Intervals

| Fill Level | Check Frequency |
|------------|-----------------|
| Up to 75% | Every 30 minutes |
| 75-85% | Every 15 minutes |
| 85-95% | Every 10 minutes |
| >95% | Continuous |

### Flow Rate Reduction

Request reduced flow rate when:
- Tank approaching 80% capacity
- Switching tanks
- Any concern about system behavior
- Valve line-up changes

## Emergency Stop Procedures

### Signals

| Signal | Meaning |
|--------|---------|
| **Radio: "STOP STOP STOP"** | Immediate emergency stop |
| **Air horn (repeated)** | Emergency stop if radio fails |
| **Red flag** | Visual emergency signal |

### Actions on Emergency Stop

| Step | Action |
|------|--------|
| 1 | Close vessel manifold valves |
| 2 | Notify bunker barge to stop |
| 3 | Identify cause |
| 4 | If spill, initiate SOPEP |
| 5 | Do not resume until cause resolved |

---

## Spill Prevention

### Scupper Plugging

All deck scuppers must be **plugged before bunkering** to prevent any spilled oil reaching the water.

### Drip Trays

| Location | Requirement |
|----------|-------------|
| Under manifold connections | Large drip tray |
| Under sounding pipes | Smaller trays |
| Under air pipes | Smaller trays |

### Absorbent Materials

Keep ready at bunkering station:
- Absorbent pads (minimum 20)
- Absorbent boom (minimum 10m)
- Shovels and buckets

## Post-Bunkering

### Completion Checklist

| Item | Checked |
|------|---------|
| Final quantities confirmed | ☐ |
| Tanks sounded, recorded | ☐ |
| Bunker delivery receipt (BDR) signed | ☐ |
| Fuel sample retained | ☐ |
| Hoses drained and disconnected | ☐ |
| Manifold blanked | ☐ |
| Deck cleaned of any drips | ☐ |
| Scuppers unplugged | ☐ |
| Oil Record Book updated | ☐ |

### Fuel Sample Retention

| Requirement | Details |
|-------------|---------|
| Sample size | Minimum 400ml |
| Labeling | Date, port, supplier, vessel |
| Storage | Sealed, kept for 12 months |
| Purpose | Dispute resolution, quality issues |
