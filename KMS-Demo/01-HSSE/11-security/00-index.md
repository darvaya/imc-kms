# Security (ISPS Code)

This section covers maritime security procedures in compliance with the International Ship and Port Facility Security (ISPS) Code.

---

## What is the ISPS Code?

The **International Ship and Port Facility Security (ISPS) Code** is a comprehensive security framework for ships and port facilities, implemented after 9/11 to prevent terrorism and unlawful acts.

---

## Key Roles

| Role | Responsibility |
|------|----------------|
| **Company Security Officer (CSO)** | Shore-based security management |
| **Ship Security Officer (SSO)** | On-board security implementation |
| **Port Facility Security Officer (PFSO)** | Port security |

---

## Security Levels

| Level | Condition | Measures |
|-------|-----------|----------|
| **Level 1** | Normal | Minimum security measures |
| **Level 2** | Heightened | Additional protective measures |
| **Level 3** | Exceptional | Further specific protective measures |

Security level is set by Flag State or Port State, not the vessel.

## ISSC Certificate

Every vessel must have an **International Ship Security Certificate (ISSC)** issued by Flag State or Recognized Security Organization.

| Detail | Requirement |
|--------|-------------|
| Validity | 5 years |
| Verification | Intermediate verification |
| On board | Original or certified copy |
