# Ship Security Plan (SSP)

## Purpose

This document provides an overview of Ship Security Plan implementation as required by the ISPS Code.

---

## Scope

Applies to: **All vessels subject to ISPS Code**

---

## What is the SSP?

The **Ship Security Plan (SSP)** is a confidential document that outlines security measures and procedures for each vessel. Key elements:

- Security organization
- Ship/port interface procedures
- Restricted area identification
- Security measures for each security level
- Security training requirements
- Communication procedures

> ⚠️ **CONFIDENTIAL:** The SSP is a confidential document. Access is restricted to authorized personnel.

---

## Security Organization

| Role | Responsibilities |
|------|------------------|
| **Company Security Officer (CSO)** | Develop and maintain SSP, security training, liaison with authorities |
| **Ship Security Officer (SSO)** | Implement SSP on board, security rounds, training crew |
| **Security Personnel** | Access control, watchkeeping, screening |

---

## Security Levels Summary

### Level 1 – Normal

| Measure | Details |
|---------|---------|
| Access control | Check ID of visitors |
| Gangway watch | Continuous attendance |
| Restricted areas | Clearly marked, controlled |
| Cargo monitoring | Supervision of cargo operations |
| Security rounds | Regular patrols |

### Level 2 – Heightened

All Level 1 measures plus:

| Measure | Details |
|---------|---------|
| Enhanced access control | Escort visitors |
| Increased frequency | More frequent rounds |
| Additional personnel | Extra security watch |
| Reduced access points | Limit entry points |
| Enhanced screening | Personal/baggage checks |

### Level 3 – Exceptional

All Level 1 and 2 measures plus:

| Measure | Details |
|---------|---------|
| Compliance with authorities | Follow government instructions |
| Full search | 100% screening |
| Restrict access | Essential personnel only |
| May include | Cessation of cargo operations |

---

## Key SSP Elements

### Access Control

| Checkpoint | Measures |
|------------|----------|
| Gangway | 24-hour watch, ID check, visitor log |
| Shell doors | Closed and locked when not in use |
| Accommodation | Controlled access |
| Engine room | Controlled access |
| Bridge | Controlled access |

### Restricted Areas

| Area | Access |
|------|--------|
| Bridge | Authorized only |
| Engine Control Room | Authorized only |
| Cargo Control Room | Authorized only |
| Communications room | Authorized only |
| SSP storage | SSO and Master only |

### Declaration of Security (DOS)

A DOS is required when:
- Security level mismatch (ship/port)
- High-risk cargo operations
- Requested by ship or port

## Security Training

| Training | For Whom | Frequency |
|----------|----------|-----------|
| Security awareness | All crew | Initial, refresher |
| Security duties | Designated personnel | Initial, refresher |
| SSO course | Ship Security Officer | Initial |
| Drills | All crew | Quarterly |

---

## Security Drills

| Drill Type | Frequency |
|------------|-----------|
| Security drill | Quarterly |
| Drill with port facility | Annually (when possible) |

### Drill Elements

- Change of security level
- Bomb threat response
- Unauthorized access response
- Evacuation procedures

---

## Reporting Security Incidents

| Incident | Report To |
|----------|-----------|
| Suspicious activity | SSO immediately |
| Breach of security | SSO → CSO → Authorities |
| Threat received | SSO → CSO → Authorities immediately |

Emergency contact: [Flag State security authority number]
