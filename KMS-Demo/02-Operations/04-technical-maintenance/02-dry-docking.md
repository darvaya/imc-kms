# Dry Docking

## Purpose

This document covers dry docking procedures for IMC Pelita Logistik vessels.

---

## What is Dry Docking?

Dry docking involves taking a vessel out of the water for:
- Underwater hull inspection and maintenance
- Propeller and rudder work
- Sea valve overhauls
- Classification surveys
- Major repairs

---

## Dry Dock Schedule

| Survey Type | Interval |
|-------------|----------|
| **Intermediate Survey** | 2.5 years ± 6 months |
| **Special Survey** | 5 years |
| **Bottom survey** | Every 2.5 years (in water or dry dock) |

## Work Specification

### Mandatory Work

| Item | Requirement |
|------|-------------|
| Hull cleaning | Remove marine growth |
| Hull painting | Antifouling, boot-top |
| Anodes | Inspect, replace |
| Propeller | Inspect, polish |
| Rudder | Inspect, bearing clearances |
| Sea valves | Overhaul |
| Sea chests | Clean, inspect |
| Stern tube | Inspect, seal |
| Tailshaft | Survey if due |

### Optional Work

| Item | Based On |
|------|----------|
| Hull repairs | Condition |
| Steelwork | Thickness surveys |
| Tank coating | Inspection results |
| Major overhauls | PMS requirements |

## Post-Dry Dock

### Before Floating

| Check | Verified |
|-------|----------|
| All sea openings closed | ☐ |
| Sea valves operational | ☐ |
| Rudder free to move | ☐ |
| Propeller clear | ☐ |
| Hull plugs removed | ☐ |

### After Floating

| Activity | Responsibility |
|----------|----------------|
| Stability check | Master |
| Engine trials | C/E |
| Steering trials | Master |
| All equipment tested | All officers |

### Documentation

| Document | Purpose |
|----------|---------|
| Class survey reports | Certification |
| Work completion certificates | Warranty |
| Photographs | Record of condition |
| Updated certificates | Renewed by class |
