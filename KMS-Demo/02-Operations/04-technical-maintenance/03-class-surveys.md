# Class Surveys

## Purpose

This document covers classification society survey requirements for IMC Pelita Logistik vessels.

---

## What is Classification?

Classification societies set and verify technical standards for ships. A classed vessel:
- Meets construction and maintenance standards
- Is surveyed regularly
- Holds valid class certificates
- Can obtain insurance and trade internationally

---

## Classification Societies

| Society | Abbreviation |
|---------|--------------|
| Biro Klasifikasi Indonesia | **BKI** |
| Lloyd's Register | **LR** |
| Det Norske Veritas | **DNV** |
| Others | As applicable |

---

## Survey Types

### Annual Survey

| Details | Information |
|---------|-------------|
| **Frequency** | Yearly (within window) |
| **Scope** | General condition, safety equipment |
| **Duration** | 1-2 days |
| **Location** | Afloat |

### Intermediate Survey

| Details | Information |
|---------|-------------|
| **Timing** | 2nd or 3rd annual |
| **Scope** | More detailed than annual |
| **Includes** | Underwater inspection |
| **Location** | Afloat or dry dock |

### Special Survey (Class Renewal)

| Details | Information |
|---------|-------------|
| **Frequency** | Every 5 years |
| **Scope** | Comprehensive |
| **Includes** | Dry dock, thickness measurements |
| **Duration** | Several days |
| **Location** | Dry dock |

### Bottom Survey

| Details | Information |
|---------|-------------|
| **Frequency** | Every 2.5 years |
| **Scope** | Underwater hull, propeller, rudder |
| **Method** | Dry dock or in-water survey |

### Tailshaft Survey

| Details | Information |
|---------|-------------|
| **Frequency** | Every 5 years |
| **Scope** | Propeller shaft withdrawal |
| **Can defer** | With approved monitoring |

---

## Continuous Survey

Some vessels operate under Continuous Survey:
- Machinery items surveyed throughout cycle
- Spreads survey workload
- Requires careful planning

---

## Survey Preparation

### Before Survey

| Action | Responsibility |
|--------|----------------|
| Review due items | Superintendent |
| Prepare documentation | Master/C/E |
| Equipment ready for inspection | Ship staff |
| Defects reported | Master |
| Surveyor appointment | Superintendent |

### Required Documents

| Document | Availability |
|----------|--------------|
| Class certificates | On board |
| Previous survey reports | On board |
| PMS records | Up to date |
| Drawings/plans | On board |
| Equipment manuals | On board |

### Equipment Testing

| Equipment | Test Required |
|-----------|---------------|
| Steering gear | Function test |
| Emergency generator | Start and load test |
| Lifeboats/davits | Launch/recovery |
| Fire pumps | Pressure test |
| Bilge pumps | Function test |

---

## During Survey

| Do | Don't |
|----|-------|
| Be cooperative | Hide defects |
| Have documents ready | Make excuses |
| Demonstrate equipment | Refuse access |
| Be honest about condition | Argue with surveyor |
| Take notes | Ignore recommendations |

---

## Survey Outcomes

| Outcome | Meaning |
|---------|---------|
| **Passed** | Satisfactory condition |
| **Recommendation** | Action suggested, not mandatory |
| **Condition of Class** | Must be rectified by deadline |
| **Detention** | Cannot sail until fixed |
| **Class suspension** | Serious, requires immediate action |

---

## Conditions of Class

| Priority | Timeframe |
|----------|-----------|
| Immediate | Before sailing |
| Short-term | Within 3 months |
| Before next survey | By next annual |
| At special survey | At next SS |
