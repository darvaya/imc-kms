# Chartering

## Purpose

This document provides an overview of charter arrangements for IMC Pelita Logistik operations.

---

## Charter Types

### Contract of Affreightment (COA)

| Characteristic | Details |
|----------------|---------|
| **Definition** | Agreement to carry specified quantity over time |
| **Volume** | Total tonnes over contract period |
| **Rate** | Per tonne |
| **Risk** | Shipowner provides vessels |
| **Common for** | Long-term coal supply contracts |

### Time Charter

| Characteristic | Details |
|----------------|---------|
| **Definition** | Vessel hired for a period |
| **Duration** | Months to years |
| **Rate** | Daily hire rate |
| **Costs (Owner)** | Crew, maintenance, insurance |
| **Costs (Charterer)** | Fuel, port costs, cargo handling |

### Voyage Charter

| Characteristic | Details |
|----------------|---------|
| **Definition** | Single voyage or shipment |
| **Rate** | Per tonne or lump sum |
| **Costs (Owner)** | All operating costs |
| **Costs (Charterer)** | Loading/discharge |

---

## Key Charter Terms

### Laytime

Time allowed for loading/discharge:

| Term | Meaning |
|------|---------|
| **Laytime** | Allowed time for cargo operations |
| **NOR** | Notice of Readiness |
| **Commencement** | When laytime starts counting |
| **Reversible** | Loading and discharge time combined |

### Demurrage & Dispatch

| Term | Definition |
|------|------------|
| **Demurrage** | Payment for time exceeding laytime |
| **Dispatch** | Rebate for completing early |
| **Rate** | $ per day (demurrage usually 2x dispatch) |

### Common Abbreviations

| Abbreviation | Meaning |
|--------------|---------|
| **SHINC** | Sundays and Holidays Included |
| **SHEX** | Sundays and Holidays Excluded |
| **WWD** | Weather Working Day |
| **PWWD** | Per Weather Working Day |
| **WIBON** | Whether In Berth Or Not |

| **Statement of Facts** | Loading/discharge timeline |

---

## Load Rate Guarantee

Many charters include loading rate guarantees:

| Component | Details |
|-----------|---------|
| **Guaranteed rate** | Tonnes per day |
| **Calculation** | Weather working days |
| **Penalty** | Demurrage if not achieved |
| **Bonus** | Dispatch if exceeded |

## Operational Implications

| Commercial Term | Operational Impact |
|-----------------|-------------------|
| **Laycan** | Vessel must arrive within window |
| **Loading rate** | FLF must maintain rate |
| **Laytime** | Track all stoppages |
| **Weather** | Document weather delays |
