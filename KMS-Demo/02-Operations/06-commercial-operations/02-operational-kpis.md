# Operational KPIs

## Purpose

This document defines key performance indicators for measuring operational performance.

---

## Why Measure?

> **"What gets measured gets managed"**

KPIs help us:
- Track performance
- Identify problems
- Benchmark against targets
- Improve continuously

---

## Key Performance Indicators

### Loading Performance

| KPI | Definition | Target |
|-----|------------|--------|
| **Gross loading rate** | Total tonnes ÷ Total time | Per contract |
| **Net loading rate** | Total tonnes ÷ Working time | Higher than gross |
| **Utilization** | Working time ÷ Total time | >85% |

### Time Performance

| KPI | Definition | Target |
|-----|------------|--------|
| **Dispatch time** | NOR to completion | Minimize |
| **Barge turnaround** | Arrival to departure | Minimize |
| **MV waiting time** | Arrival to berth | Minimize |

### Commercial Performance

| KPI | Definition | Target |
|-----|------------|--------|
| **Demurrage** | Days over laytime | Zero |
| **Dispatch earned** | Days saved | Maximize |
| **On-time delivery** | Within laycan | 100% |

### Fleet Performance

| KPI | Definition | Target |
|-----|------------|--------|
| **Availability** | Time available ÷ Total time | >95% |
| **Off-hire** | Time not available | <5% |
| **Breakdown** | Unplanned stoppages | Minimize |

### Safety Performance

| KPI | Definition | Target |
|-----|------------|--------|
| **LTIFR** | Lost time injuries per million man-hours | <0.5 |
| **TRIR** | Total recordable incidents | Minimize |
| **Near misses** | Reported near misses | Increase (positive) |
| **PSC detentions** | Port State detentions | Zero |

## Measurement & Reporting

### Data Collection

| Source | Data |
|--------|------|
| **Vessel reports** | Daily reports |
| **FLF logs** | Loading records |
| **Surveyor reports** | Quantities |
| **Statement of Facts** | Timelines |

### Reporting Frequency

| Report | Frequency | To |
|--------|-----------|-----|
| Daily report | Daily | Operations Manager |
| Weekly summary | Weekly | Management |
| Monthly review | Monthly | Board |
| Quarterly review | Quarterly | Stakeholders |

## Stoppage Categories

Track stoppages by cause:

| Category | Examples |
|----------|----------|
| **Weather** | Wind, swell, rain |
| **Equipment** | Crane, conveyor breakdown |
| **Vessel** | MV issues |
| **Operational** | Barge shortage, shift change |
| **Other** | Port authority, force majeure |
