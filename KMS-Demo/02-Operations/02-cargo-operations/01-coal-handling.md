# Coal Handling

## Purpose

This document covers the handling requirements for coal cargo transported by IMC Pelita Logistik.

---

## Coal Types Transported

| Type | Characteristics | Handling Notes |
|------|-----------------|----------------|
| **Thermal Coal** | Power generation, high volume | Standard handling |
| **Sub-bituminous** | Lower rank, moisture content | Self-heating risk |
| **Coking Coal** | Metallurgical use | Less common |

---

## Coal Properties Summary

| Property | Impact |
|----------|--------|
| **Calorific value** | Commercial quality measure |
| **Moisture content** | Affects self-heating, weight |
| **Ash content** | Quality parameter |
| **Volatile matter** | Flammability risk |
| **Sulfur content** | Quality, environmental concern |

For detailed safety information, see [Coal Properties & Hazards](../../01-HSSE/04-coal-cargo-safety/01-coal-properties-hazards.md).

---

## Handling Requirements

### General Principles

| Principle | Details |
|-----------|---------|
| **Minimize handling** | Each transfer degrades quality |
| **Dust control** | Suppress dust during transfers |
| **Segregation** | Keep different grades separate |
| **Contamination** | Avoid mixing with other cargoes |
| **Documentation** | Track quality throughout chain |

### Loading

| Requirement | Details |
|-------------|---------|
| Barge cleanliness | Holds swept before loading |
| Loading method | Conveyor preferred |
| Trimming | Even distribution |
| Sampling | Per contract requirements |
| Draft survey | Before and after |

### Transport

| Requirement | Details |
|-------------|---------|
| Coverage | Open deck acceptable |
| Bilge | Keep bilges pumped |
| Weather | Monitor rainfall impact |
| Duration | Minimize time afloat |

### Transshipment

| Requirement | Details |
|-------------|---------|
| Grab type | Clamshell grab |
| Spillage | Minimize, clean deck |
| Rate | Maintain steady rate |
| Quality | No mixing of grades |

## Cargo Declaration (IMSBC)

Under the IMSBC Code, shippers must provide:

| Information | Purpose |
|-------------|---------|
| Cargo name | Correct identification |
| Moisture content | Liquefaction risk (not for coal) |
| Transportable moisture limit | If applicable |
| Hazard characteristics | Self-heating, gas emission |
| Handling instructions | Specific requirements |
