# Discharge to MV

## Purpose

This document describes procedures for loading (discharging) coal from FLF to motor vessels.

---

## Overview

The final stage of transshipment involves loading coal onto the mother vessel (MV) for ocean transport.

---

## Pre-Loading

### MV Preparation

| Requirement | Details |
|-------------|---------|
| Holds clean | Swept, free of previous cargo |
| Bilges | Clear, covers in place |
| Ventilation | Lids open during loading |
| Hatches | Open and secured |
| Stability | Ballast plan prepared |

### Pre-Loading Meeting

| Attendees | Topics |
|-----------|--------|
| Master/C/O | Stowage plan, hatch sequence |
| FLF Foreman | Loading rate, boom positioning |
| Surveyor | Survey procedures |

---

## Stowage Plan

### Considerations

| Factor | Consideration |
|--------|---------------|
| **Stability** | Maintain positive GM throughout |
| **Trim** | Safe trim during and after loading |
| **Stress** | Bending/shearing within limits |
| **Hatch sequence** | Cargo distribution |
| **Draft** | Do not exceed marks |

### Typical Loading Sequence

| Principle | Action |
|-----------|--------|
| **Start** | Load multiple hatches spread out |
| **Balance** | Rotate between hatches |
| **Ballast** | De-ballast as cargo loaded |
| **Finish** | Final trim hatches |

---

## Loading Operations

### Boom Positioning

| Factor | Details |
|--------|---------|
| **Reach** | Position for hatch access |
| **Movement** | Radial swing within hatch |
| **Coverage** | Even distribution in hold |

### Hold Filling

| Phase | Action |
|-------|--------|
| **Initial** | Fill bottom, work outward |
| **Building** | Layer cargo evenly |
| **Trimming** | Final leveling for capacity |

### Rate Coordination

| Party | Responsibility |
|-------|----------------|
| **FLF** | Maintain steady loading rate |
| **MV** | Monitor hold levels, request stops |
| **Both** | Coordinate hatch changes |

---

## Ballast Operations

### During Loading

| Action | Responsibility |
|--------|----------------|
| Monitor drafts | MV C/O |
| Calculate cargo | Surveyor/C/O |
| Adjust ballast | MV C/O |
| Communicate changes | MV to FLF |

### Ballast Discharge

| Requirement | Details |
|-------------|---------|
| Location | Per MARPOL regulations |
| Documentation | Ballast Water Record Book |
| BWM System | Use if required |

---

## Hatch Changeover

| Step | Action |
|------|--------|
| 1 | FLF stops loading |
| 2 | Boom slewed to next hatch |
| 3 | MV confirms ready |
| 4 | Loading resumes |

---

## Cargo Monitoring

### During Loading

| Monitor | Frequency |
|---------|-----------|
| Draft readings | Each hatch |
| Cargo temperature | As received |
| Loading rate | Continuous |
| Stability | Continuous |

### Cargo Care

| Concern | Action |
|---------|--------|
| **Dust** | Water spray if excessive |
| **Spillage** | Clean deck |
| **Damage** | Report any issues |

---

## Completion

### Final Loading

| Step | Action |
|------|--------|
| 1 | Final trimming in hatches |
| 2 | Loading stopped |
| 3 | Boom secured |
| 4 | Final draft readings |

### Draft Survey

| Survey | Details |
|--------|---------|
| Readings | All six draft marks |
| Corrections | Trim, list, deformation |
| Calculation | Loaded quantity |
| Certificate | Issued by surveyor |

### Hold Inspection

| Check | Requirement |
|-------|-------------|
| Cargo level | Properly trimmed |
| No damage | Holds undamaged |
| Ready for sea | Ventilation set |
