# Transshipment Workflow

## Purpose

This document describes the complete transshipment workflow at the Floating Loading Facility (FLF).

---

## Transshipment Overview

Transshipment is the transfer of coal from barges to ocean-going vessels (MVs) at anchorage using the FLF.

```
         Barge Arrival            FLF Operations         MV Loading
              │                         │                    │
              ▼                         ▼                    ▼
┌──────────────────────────────────────────────────────────────────┐
│   BARGE ───► MOORING ───► DISCHARGE ───► CONVEYOR ───► MV HOLD  │
└──────────────────────────────────────────────────────────────────┘
```

---

## Workflow Phases

### Phase 1: Pre-Arrival

| Actor | Action |
|-------|--------|
| **Operations** | Confirm MV and barge ETAs |
| **FLF** | Prepare for MV berthing |
| **Tug** | Confirm barge convoy |
| **Surveyor** | Arrange for surveys |

### Phase 2: MV Berthing

| Step | Duration | Action |
|------|----------|--------|
| 1 | - | MV approaches FLF |
| 2 | 1-2 hrs | Mooring operations |
| 3 | 1 hr | Pre-loading meeting |
| 4 | 1-2 hrs | Draft survey (initial) |
| 5 | - | Hatches opened, ready |

### Phase 3: Barge Queue

| Step | Action |
|------|--------|
| 1 | Barges arrive and anchor |
| 2 | Queue established by arrival |
| 3 | First barge called to berth |

### Phase 4: Barge Discharge Cycle

| Step | Action |
|------|--------|
| 1 | Tug positions barge alongside FLF |
| 2 | Mooring lines secured |
| 3 | Crane dischages barge to hopper |
| 4 | Conveyor transfers to MV |
| 5 | Barge emptied |
| 6 | Barge cast off, next barge moors |
| 7 | Repeat until MV loaded |

### Phase 5: MV Completion

| Step | Duration | Action |
|------|----------|--------|
| 1 | 1 hr | Final trimming |
| 2 | 1-2 hrs | Draft survey (final) |
| 3 | 1 hr | Hatch closing |
| 4 | - | Documents signed |
| 5 | 1 hr | MV casts off |

### Communication

| Parties | Communication |
|---------|---------------|
| FLF – MV | VHF, face-to-face |
| FLF – Barge/Tug | VHF |
| FLF – Shore | Phone, email |
| Surveyor – All | On-site |

### Meetings

| Meeting | When | Purpose |
|---------|------|---------|
| Pre-loading | MV berthing | Agree stowage plan, procedures |
| Shift handover | Each shift | Status, issues, forecast |
| Daily | Morning | Operations planning |

## Barge Management

### Queue Priority

| Priority | Criteria |
|----------|----------|
| 1 | Arrival time (first come, first served) |
| 2 | Contract/customer priority |
| 3 | Cargo grade matching |

### Empty Barge Departure

| Step | Action |
|------|--------|
| 1 | Discharge complete |
| 2 | Barge inspected for residue |
| 3 | Cast off |
| 4 | Tug moves to holding area |
| 5 | Return voyage to jetty |
