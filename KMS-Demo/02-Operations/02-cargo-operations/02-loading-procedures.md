# Loading Procedures

## Purpose

This document describes the procedures for loading coal onto barges at mine loading jetties.

---

## Loading Overview

Coal is loaded from mine stockpiles onto barges at loading jetties along rivers in Kalimantan.

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   STOCKPILE  │────►│   CONVEYOR   │────►│    BARGE     │
│              │     │    SYSTEM    │     │              │
└──────────────┘     └──────────────┘     └──────────────┘
```

---

## Pre-Loading

### Barge Preparation

| Check | Requirement |
|-------|-------------|
| Holds | Clean, swept, no residue |
| Bilges | Empty, pumps operational |
| Structure | No damage, watertight |
| Draft marks | Visible, verified |
| Documentation | Ready for survey |

### Jetty Coordination

| Step | Action |
|------|--------|
| 1 | Confirm loading window |
| 2 | Advise coal grade/quantity |
| 3 | Arrange draft surveyor |
| 4 | Confirm barge position |

---

## Loading Sequence

| Phase | Activity |
|-------|----------|
| **Arrival** | Tug positions barge alongside jetty |
| **Mooring** | Secure with mooring lines |
| **Initial survey** | Draft survey before loading |
| **Loading** | Coal loaded via conveyor/chute |
| **Monitoring** | Draft, trim, and rate |
| **Completion** | Loading stopped at target |
| **Final survey** | Draft survey after loading |
| **Documentation** | Bill of lading, cargo manifest |
| **Departure** | Cast off, proceed to anchorage |

---

## Loading Methods

### Conveyor/Chute System

| Feature | Details |
|---------|---------|
| Method | Fixed or movable conveyor |
| Rate | 500-2,000 tonnes/hour |
| Control | Operator at jetty |
| Trim | Move barge along jetty |

### Grab/Crane Loading

| Feature | Details |
|---------|---------|
| Method | Shore crane with grab |
| Rate | Lower than conveyor |
| Use | Smaller operations |

---

## Loading Rate Considerations

| Factor | Impact |
|--------|--------|
| **Conveyor capacity** | Maximum rate |
| **Barge size** | Total duration |
| **Trim requirements** | May require slow loading |
| **Tide/current** | May restrict operations |
| **Weather** | Rain may affect quality |

---

## Draft and Trim

### Draft Monitoring

| Action | Frequency |
|--------|-----------|
| Read drafts | Every 30 minutes |
| Record readings | Port/starboard, fore/aft |
| Calculate cargo | Progressive quantity |

### Maximum Draft

| Consideration | Factor |
|---------------|--------|
| River depth | Shallow sections on route |
| Tide | Low water clearance |
| Barge limits | Maximum design draft |
| Freeboard | Minimum required |

### Trim Adjustment

| Method | Action |
|--------|--------|
| Move barge | Shift along jetty |
| Vary loading | Adjust loading point |
| Stop and shift | Major trim correction |

## Documentation

| Document | Purpose |
|----------|---------|
| **Draft survey** | Quantity determination |
| **Bill of lading** | Title to cargo |
| **Cargo manifest** | Details of cargo |
| **Quality certificate** | From loading point |

## Post-Loading

| Check | Action |
|-------|--------|
| Final trim | Within acceptable limits |
| Freeboard | Adequate for voyage |
| Lashing | Secure (if applicable) |
| Documentation | Complete and signed |
| Weather | Suitable for departure |
