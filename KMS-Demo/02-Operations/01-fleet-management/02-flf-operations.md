# FLF Operations

## Purpose

This document describes the operational procedures for Floating Loading Facilities (FLF) in the transshipment of coal from barges to mother vessels.

---

## FLF Operational Overview

The FLF is the core of the transshipment operation – receiving coal from barges and loading it onto ocean-going vessels for export.

```
                        ┌────────────────┐
                        │  MOTHER VESSEL │
                        │    (MV)        │
                        └───────▲────────┘
                                │ Loading boom
                        ┌───────┴────────┐
   ┌────────┐          │                 │
   │ BARGE  │──────────│      FLF       │
   │        │  Crane   │                 │
   └────────┘          └─────────────────┘
```

## Daily Operations

### Pre-Operation Checks

| Check | Responsibility | Frequency |
|-------|----------------|-----------|
| Crane function test | Crane operator | Each shift |
| Conveyor inspection | Deck crew | Each shift |
| Mooring equipment | Bosun | Daily |
| Fire-fighting equipment | Safety officer | Daily |
| Communication systems | Radio operator | Each shift |
| Weather conditions | OOW | Continuous |

### Operational Sequence

| Phase | Activity | Duration |
|-------|----------|----------|
| 1. Pre-arrival | Confirm barge/MV arrival time | - |
| 2. MV berthing | MV approaches and moors | 2-4 hours |
| 3. Pre-survey | Draft survey, hold inspection | 1-2 hours |
| 4. Barge arrival | First barge moors alongside | 1 hour |
| 5. Loading | Continuous loading cycles | Days |
| 6. Completion | Final survey, MV departure | 2-4 hours |

## Barge Handling

### Barge Approach

| Step | Action |
|------|--------|
| 1 | Confirm barge ETA via radio |
| 2 | Prepare mooring lines and fenders |
| 3 | Tug positions barge alongside FLF |
| 4 | Pass mooring lines |
| 5 | Secure barge, adjust for crane reach |

### Barge Discharge Cycle

| Step | Action | Time |
|------|--------|------|
| 1 | Position crane over barge hold | 2 min |
| 2 | Lower grab, fill grab | 30 sec |
| 3 | Lift grab, swing to hopper | 30 sec |
| 4 | Discharge into hopper | 15 sec |
| 5 | Return for next grab | 30 sec |
| **Cycle time** | Per grab | ~2 min |

### Barge Changeover

| Step | Action |
|------|--------|
| 1 | Complete discharge of current barge |
| 2 | Confirm next barge ready |
| 3 | Cast off current barge |
| 4 | Tug moves empty barge away |
| 5 | Tug positions loaded barge |
| 6 | Moor and resume loading |

## MV Loading

### Hatch Sequence

| Consideration | Factor |
|---------------|--------|
| **Sequence** | As per Chief Officer's plan |
| **Rotation** | Ballast adjustment between hatches |
| **Trim** | Maintain safe trim during loading |
| **Stress** | Monitor hull bending moments |

### Loading Rate Control

| Factor | Action |
|--------|--------|
| **Conveyor speed** | Adjust to match grab rate |
| **Crane rate** | Maintain steady grab cycle |
| **MV ballast** | Coordinate with MV C/O |
| **Hatch trimming** | Trim cargo as required |

## Weather Limits

### Operational Limits

| Condition | Limit | Action |
|-----------|-------|--------|
| Wind | >25 knots | Suspend operations |
| Wave height | >2 meters | Suspend operations |
| Visibility | <0.5 nm | Suspend crane ops |
| Lightning | Within 5 nm | Suspend operations |

### Weathering Position

If weather exceeds limits:
1. Secure all cargo gear
2. Cast off barges (if safer)
3. MV may cast off or remain moored
4. FLF maintains position (anchored/moored)
