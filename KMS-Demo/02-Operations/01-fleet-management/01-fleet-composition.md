# Fleet Composition

## Purpose

This document provides specifications and details of IMC Pelita Logistik's vessel fleet.

---

## Floating Loading Facilities (FLF)

FLFs are purpose-built transshipment platforms for transferring coal from barges to ocean-going vessels.

### Typical FLF Specifications

| Specification | Details |
|---------------|---------|
| **Type** | Semi-submersible or barge-based |
| **Length** | 100-150 meters |
| **Beam** | 30-40 meters |
| **Loading Rate** | 1,500-3,000 tonnes/hour |
| **Crane Capacity** | 25-35 tonne grabs |
| **Conveyor System** | Belt conveyors with radial stacker |
| **Holding Capacity** | Limited (pass-through operation) |
| **Crew** | 20-30 personnel |

### FLF Features

| Feature | Purpose |
|---------|---------|
| Crane(s) | Grab coal from barges |
| Hopper | Receive coal from crane |
| Conveyor | Transfer to loading boom |
| Loading boom | Radial for MV hatch coverage |
| Mooring dolphins | Secure barges and MVs |
| Accommodation | Crew quarters |
| Power generation | Self-contained power |

---

## Tugs

Tugs provide motive power for barges throughout river and coastal operations.

### Harbor/River Tugs

| Specification | Details |
|---------------|---------|
| **Type** | Twin-screw or ASD |
| **Length** | 25-35 meters |
| **Bollard Pull** | 25-50 tonnes |
| **Power** | 1,500-3,000 HP |
| **Role** | Barge towing in rivers and anchorage |
| **Crew** | 6-10 personnel |

### Ocean-Going Tugs

| Specification | Details |
|---------------|---------|
| **Type** | ASD or conventional |
| **Length** | 35-45 meters |
| **Bollard Pull** | 50-80 tonnes |
| **Power** | 3,000-5,000 HP |
| **Role** | Long-distance towing, emergency assist |
| **Crew** | 8-12 personnel |

---

## Barges

Barges are unpowered cargo vessels carrying coal from loading jetties to FLFs.

### Coal Barges

| Specification | Details |
|---------------|---------|
| **Type** | Flat-deck bulk barge |
| **Length** | 80-100 meters |
| **Beam** | 20-24 meters |
| **Depth** | 5-6 meters |
| **Capacity** | 8,000-12,000 DWT |
| **Draft (loaded)** | 4-5 meters |
| **Construction** | Steel, double-skin |
| **Hatches** | Open deck (no covers) |
| **Crew** | Unmanned (tug provides crew) |

### Barge Size Categories

| Category | Capacity | Use |
|----------|----------|-----|
| **Small** | 5,000-8,000 DWT | Shallow river sections |
| **Medium** | 8,000-10,000 DWT | Standard river ops |
| **Large** | 10,000-12,000 DWT | Deep river/anchorage |

---

## Motor Vessels (MVs)

MVs are ocean-going bulk carriers that receive coal for export.

### Typical MV Sizes (Chartered)

| Size | DWT Range | Hatches | Use |
|------|-----------|---------|-----|
| **Handysize** | 25,000-40,000 | 4-5 | Short-haul |
| **Supramax** | 50,000-60,000 | 5-6 | Common size |
| **Panamax** | 70,000-80,000 | 7 | Larger shipments |
| **Capesize** | 150,000+ | 9 | Major exports |
