# Tug & Barge Operations

## Purpose

This document describes the operational procedures for tugs and barges in the coal transport chain from loading jetties to transshipment points.

---

## Tug & Barge Overview

Tugs and barges form the backbone of the river transport system, moving coal from mine loading points to FLF transshipment areas.

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  LOADING    │────►│   RIVER     │────►│  ANCHORAGE  │
│   JETTY     │     │  TRANSPORT  │     │    (FLF)    │
└─────────────┘     └─────────────┘     └─────────────┘
   Loading          Tug + Barge            Discharge
```

## Tug Operations

### Pre-Departure Checklist

| Check | Responsibility |
|-------|----------------|
| Engine room checks | Chief Engineer |
| Navigation equipment | Master |
| Towing gear inspection | Chief Officer |
| Fuel and water levels | Chief Engineer |
| Weather conditions | Master |
| Barge secured properly | Chief Officer |

### Towing Configurations

| Configuration | Use Case |
|---------------|----------|
| **Astern towing** | River transit, open water |
| **Alongside** | Maneuvering, berthing |
| **Push mode** | Short distances, tight spaces |

### River Transit Procedures

| Phase | Procedure |
|-------|-----------|
| **Departure** | Cast off, clear jetty, stream towline |
| **Transit** | Maintain safe speed, monitor current |
| **Passing** | Coordinate with traffic (VHF) |
| **Arrival** | Shorten towline, shift to alongside |
| **Berthing** | Position barge, pass mooring lines |

### Speed Guidelines

| Area | Speed Limit |
|------|-------------|
| Jetty area | Dead slow |
| Narrow channels | Slow ahead |
| Open river | Standard (current-dependent) |
| Anchorage | Slow approach |

## Barge Operations

### Loading at Jetty

| Step | Action |
|------|--------|
| 1 | Tug positions barge alongside |
| 2 | Moor barge securely |
| 3 | Conveyor/chute loading begins |
| 4 | Monitor draft and trim |
| 5 | Complete loading, obtain draft survey |
| 6 | Prepare for departure |

### Loading Requirements

| Requirement | Details |
|-------------|---------|
| **Draft limits** | As per river depth |
| **Trim** | Even keel or slight stern trim |
| **Freeboard** | Minimum freeboard maintained |
| **Hatch covers** | Open deck (no covers on coal barges) |
| **Documentation** | Bill of lading, draft survey |

### Cargo Care During Transit

| Item | Action |
|------|--------|
| **Cargo coverage** | None required for coal |
| **Monitoring** | Visual inspection during transit |
| **Bilge pumping** | As required |
| **Weather protection** | Seek shelter if heavy rain |

## River Navigation

### River Hazards

| Hazard | Mitigation |
|--------|------------|
| **Shifting channels** | Follow updated navigation marks |
| **Shallow areas** | Draft monitoring, local knowledge |
| **Strong current** | Adjust speed, approach angles |
| **Other traffic** | VHF coordination, passing agreements |
| **Floating debris** | Lookout, reduced speed |
| **Low bridges** | Air draft clearance |

### Tide/Current Considerations

| Factor | Impact |
|--------|--------|
| **Current direction** | Affects fuel consumption, ETA |
| **Current strength** | May restrict operations |
| **Tidal range** | Affects draft at shallow points |
| **Timing** | Plan transit for favorable conditions |
