# Motor Vessel Operations

## Purpose

This document describes operational procedures for Motor Vessels (MVs) receiving coal cargo at anchorage via FLF transshipment.

---

## MV Operational Overview

Motor Vessels are ocean-going bulk carriers that receive coal cargo at anchorage for export to international destinations.

---

## Pre-Arrival

### Vessel Nomination

| Step | Action |
|------|--------|
| 1 | Charterer nominates vessel |
| 2 | Vessel vetted (Rightship/RISQ) |
| 3 | Acceptance confirmed |
| 4 | Laycan agreed |
| 5 | Agent appointed |

### Pre-Arrival Information Exchange

| Information | From MV | From Shore |
|-------------|---------|------------|
| ETA | ✅ | - |
| Cargo details | - | ✅ |
| Stowage plan | ✅ | - |
| Draft requirements | ✅ | ✅ |
| FLF position | - | ✅ |

---

## Anchorage Arrival

### Approach Procedure

| Step | Action |
|------|--------|
| 1 | Contact agent/FLF via VHF |
| 2 | Receive approach instructions |
| 3 | Proceed to anchorage |
| 4 | Anchor or proceed to FLF |
| 5 | Await berthing instructions |

### Berthing at FLF

| Step | Action |
|------|--------|
| 1 | Approach FLF at slow speed |
| 2 | FLF provides mooring instructions |
| 3 | Pass mooring lines (bow, stern, springs) |
| 4 | Adjust lines, position for loading |
| 5 | Confirm all fast |

---

## Loading Operations

### Hold Preparation

| Check | Requirement |
|-------|-------------|
| Hold cleanliness | Swept, no residue |
| Bilge covers | In place, secure |
| Ladders | Secure, accessible |
| Ventilation | Lids open |
| Lighting | Functional |

### Loading Sequence

| Phase | Activity |
|-------|----------|
| **Initial survey** | Draft survey by surveyor |
| **Hatch opening** | Open hatches, deploy cranes |
| **Loading** | Per stowage plan |
| **Trimming** | Trim cargo in each hatch |
| **Ballast** | Adjust ballast during loading |
| **Final survey** | Confirm loaded quantity |

### Loading Plan Considerations

| Factor | Consideration |
|--------|---------------|
| **Sequence** | Load hatches per plan |
| **Trim** | Maintain acceptable trim |
| **Stress** | Monitor bending/shearing |
| **Stability** | Ensure GM positive |
| **Draft** | Do not exceed marks |

| Monitor | Frequency |
|---------|-----------|

## Completion & Departure

### Completion Procedure

| Step | Action |
|------|--------|
| 1 | Loading complete, confirm quantity |
| 2 | Final draft survey |
| 3 | Sign cargo documents |
| 4 | Close and secure hatches |
| 5 | Check lashing (if any) |
| 6 | Prepare for sea |

### Departure

| Step | Action |
|------|--------|
| 1 | All cargo documents received |
| 2 | Crew on board, counted |
| 3 | Master's departure checklist |
| 4 | Cast off mooring lines |
| 5 | Clear FLF, proceed to sea |

---

## Voyage Considerations

### Post-Loading Voyage

| Requirement | Details |
|-------------|---------|
| **Cargo monitoring** | Temperature, atmosphere checks |
| **Ventilation strategy** | Per coal cargo procedures |
| **Weather routing** | Avoid heavy weather if possible |
| **Hatch security** | Confirm watertight |
