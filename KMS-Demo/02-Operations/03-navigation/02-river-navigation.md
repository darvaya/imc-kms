# River Navigation

## Purpose

This document covers navigation procedures specific to Indonesian river operations.

---

## Primary Operating Rivers

| River | Location | Characteristics |
|-------|----------|-----------------|
| **Mahakam River** | East Kalimantan | Major coal transport route |
| **Barito River** | South Kalimantan | Coal transport |
| **Other rivers** | Various | Project-specific |

---

## River Navigation Characteristics

Unlike open water, river navigation involves:

| Factor | Challenge |
|--------|-----------|
| **Narrow channels** | Limited maneuvering |
| **Shallow water** | Draft restrictions |
| **Strong currents** | Variable with tide/rain |
| **Shifting channels** | Sandbanks change |
| **Traffic** | Two-way traffic in narrow waters |
| **Limited visibility** | Bends, obstructions |
| **Environmental** | Floating debris, logs |

---

## Draft Restrictions

### Controlling Depths

| Factor | Consideration |
|--------|---------------|
| **Charted depth** | May not be surveyed recently |
| **Tide** | Significant impact |
| **River level** | Rainfall affects level |
| **Season** | Dry vs wet season |

### Maximum Draft Policy

| Principle | Action |
|-----------|--------|
| Conservative | Leave margin for safety |
| Local knowledge | Rely on pilots/experienced masters |
| Real-time info | Check current river conditions |
| Contingency | Know where to anchor if stuck |

---

## Navigation Marks

### Indonesian River Marks

| Mark Type | Meaning |
|-----------|---------|
| **Port-hand buoys** | Keep to port (red) |
| **Starboard-hand buoys** | Keep to starboard (green) |
| **Cardinal marks** | Indicate safe water direction |
| **Leading marks** | Transit/leading lines |
| **Anchoring marks** | Designated areas |

> ⚠️ **Note:** River marks may not always be maintained. Never rely solely on navigation aids.

---

## Passing Procedures

### Meeting Traffic

| Situation | Action |
|-----------|--------|
| Head-on | Coordinate via VHF, pass port-to-port |
| Overtaking | Coordinate, overtaking vessel keeps clear |
| At bends | Slow down, sound signals |

### Communication

| Action | Communication |
|--------|---------------|
| Pre-passing | VHF call to agree side |
| Confirmation | Confirm intentions |
| Execution | Execute as agreed |

---

## Current Effects

### With Current (Downstream)

| Effect | Action |
|--------|--------|
| Faster SOG | Maintain steerage |
| Reduced control | Plan ahead |
| Stopping distance | Greatly increased |

### Against Current (Upstream)

| Effect | Action |
|--------|--------|
| Slower SOG | Increased fuel |
| Better control | Steering more responsive |
| Reduced stopping distance | |

---

## Bend Navigation

| Principle | Action |
|-----------|--------|
| **Slow down** | Reduce speed approaching bends |
| **Sound signal** | One prolonged blast if unsure |
| **Stay in channel** | Do not cut corners |
| **Listen** | For signals from other vessels |

---

## Night Navigation

| Requirement | Details |
|-------------|---------|
| Reduced speed | As appropriate |
| Enhanced lookout | Dedicated lookout |
| Navigation lights | Confirmed operational |
| Radar | Active use |
| Local restrictions | Some rivers restrict night transit |
