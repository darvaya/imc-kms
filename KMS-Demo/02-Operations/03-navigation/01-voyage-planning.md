# Voyage Planning

## Purpose

This document describes voyage planning procedures for all IMC Pelita Logistik vessels.

---

## Overview

Voyage planning is a legal requirement (SOLAS Chapter V, Regulation 34) and essential for safe navigation. All voyages must be planned in advance.

---

## Voyage Planning Stages

### 1. Appraisal

| Activity | Details |
|----------|---------|
| Gather information | Charts, publications, notices |
| Review route | Identify hazards, constraints |
| Weather | Obtain forecasts |
| Vessel factors | Draft, UKC, speed |
| Traffic | Expected traffic conditions |

### 2. Planning

| Activity | Details |
|----------|---------|
| Plot route | On appropriate charts |
| Mark waypoints | Course alterations |
| Calculate distances | Leg by leg |
| Identify margins | Safety margins, no-go areas |
| Note contingencies | Anchor areas, shelter |

### 3. Execution

| Activity | Details |
|----------|---------|
| Monitor progress | Continuous position fixing |
| Cross-check | Multiple sources |
| Communicate | With traffic, VTS |
| Record | Log all events |
| Adjust | If conditions change |

### 4. Monitoring

| Activity | Details |
|----------|---------|
| Track vs plan | Compare actual to planned |
| Deviations | Investigate, correct |
| Lessons learned | Feedback for improvement |

---

## Passage Plan Contents

| Element | Details |
|---------|---------|
| **Route** | Waypoints, courses, distances |
| **Charts** | Required charts listed |
| **Hazards** | Marked on charts |
| **No-go areas** | Shallow water, dangers |
| **Contingencies** | Alternative routes, anchoring |
| **Communications** | VTS, port contacts |
| **Tides/currents** | Expected conditions |
| **Weather** | Forecast summary |
| **Traffic** | Expected traffic |
| **ETAs** | For key points |

## Under-Keel Clearance (UKC)

### Minimum UKC

| Area | Minimum UKC |
|------|-------------|
| Open water | 2.0 meters or 10% draft |
| Rivers | As per local regulations |
| Port approaches | Per port requirements |
| Alongside berth | Per terminal requirements |

### UKC Calculation

| Factor | Consider |
|--------|----------|
| Charted depth | From charts |
| Tide | Height of tide |
| Squat | At speed |
| Heel/list | Effective draft increase |
| Wave action | Swell-induced motion |
