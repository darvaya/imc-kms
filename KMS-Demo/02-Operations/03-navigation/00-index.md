# Navigation

This section covers navigation procedures for voyage planning, river navigation, and coastal/anchorage operations.

---

## Navigation Principles

1. **Safety** – Safe navigation is paramount
2. **Planning** – Every voyage is planned in advance
3. **Monitoring** – Continuous position monitoring
4. **Communication** – Clear communications with traffic
5. **Compliance** – Adherence to regulations and COLREGs

## Navigation Equipment

| Equipment | Purpose |
|-----------|---------|
| GPS/DGPS | Position fixing |
| AIS | Traffic identification |
| Radar | Collision avoidance |
| ECDIS | Electronic charts (if fitted) |
| Paper charts | Backup navigation |
| Depth sounder | Under-keel clearance |
| VHF radio | Communications |

---

## Bridge Watchkeeping

All vessels follow STCW requirements for bridge watchkeeping:

| Requirement | Details |
|-------------|---------|
| OOW qualified | STCW-certified officers |
| Lookout | Dedicated in restricted visibility |
| Handover | Formal handover procedures |
| Rest hours | Per STCW work/rest |
