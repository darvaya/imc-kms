# Coastal Navigation

## Purpose

This document covers coastal navigation and anchorage operations for IMC Pelita Logistik vessels.

---

## Coastal Operations

Coastal navigation includes:
- Transits between river mouths and anchorages
- Anchorage approaches
- Operations at anchorage
- Coastal voyages

---

## Anchorage Approach

### Pre-Approach

| Action | Details |
|--------|---------|
| Confirm position | FLF/anchorage location |
| Contact FLF | VHF for approach instructions |
| Weather check | Current conditions |
| Traffic | Other vessels in area |
| Draft | Confirm safe depth |

### Approach Procedure

| Step | Action |
|------|--------|
| 1 | Reduce speed |
| 2 | Follow designated approach track |
| 3 | Maintain communication |
| 4 | Prepare mooring equipment |
| 5 | Proceed as directed |

## Anchoring

### When Required

| Situation | Action |
|-----------|--------|
| Waiting for berth | Anchor in designated area |
| Weather delay | Anchor for safety |
| Emergency | Safe anchoring |

### Anchoring Procedure

| Step | Action |
|------|--------|
| 1 | Confirm anchor position |
| 2 | Approach position into wind |
| 3 | Stop vessel |
| 4 | Let go anchor |
| 5 | Veer cable to scope |
| 6 | Check holding |
| 7 | Set anchor watch |

### Anchor Watch

| Duty | Details |
|------|---------|
| Position monitoring | Plot position regularly |
| Weather | Monitor changes |
| Traffic | Watch for vessels |
| Anchor bearing | Check for dragging |
| Communications | Monitor VHF |

---

## Weather Considerations

### Wind Limits (Anchorage Operations)

| Condition | Action |
|-----------|--------|
| <15 knots | Normal operations |
| 15-25 knots | Caution, monitor |
| >25 knots | Consider suspending ops |
| Storm warning | All vessels to sea room |

### Swell

| Condition | Action |
|-----------|--------|
| <1.5 m | Normal operations |
| 1.5-2.0 m | Caution |
| >2.0 m | Suspend loading ops |
